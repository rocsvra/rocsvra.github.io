WPF（4）：控件

发表于 2023-04-13 更新于 2023-05-05 分类于 WPF

WPF中，数据驱动UI，Control是数据和行为的载体，用来展示数据和响应用户操作。

常用控件

控件类型 共同父类 示例
布局控件 Panel Grid、StackPanel、DockPanel
内容控件 ContentControl Window、Button
带标题的内容控件 HeaderedContentControl GroupBox、TabItem
条目控件 ItemsControl ListBox、ComboBox
带标题的条目控件 HeaderedItemsControl TreeViewItem、MenuItem
特殊内容控件 - TextBox、TextBlock、Image

FrameworkElement类在UIElement类的基础上添加了很多专门用于WPF开发的API（如SetBiding方法），从这个类开始才算是进入WPF开发框架；

UI元素类型 注释
ContentControl 单一内容控件
HeaderedContentControl 带标题的单一内容控件
ItemsControl 以条目集合为内容的控件
HeaderedItemsControl 带标题的以条目集合为内容的控件
Decorator 控件装饰元素
Panel 面板类元素
Adorner 文字点缀元素
Flow Text 流式文本元素
TextBox 文本输入框
TextBlock 静态文字
Shape 图形元素

以下三种写法是等价的——

完整语法

简写

ContentControl族

Button ButtonBase CheckBox ComboBoxItem
ContentControl Frame GridViewColumnHeader GroupItem
Label ListBoxItem ListViewItem NavigationWindow
RadioButton RepeatButton ScrollViewer StatusBarItem
ToggleButton ToolTip UserControl Window

派生自ContentControl的控件，内容属性的名称为Content；
只能由单一元素充当其内容

HeaderedContentControl族

Expander GroupBox HeaderedContentControl TabItem

派生自HeaderedContentControl的控件类，HeaderedContentControl派生自ContentControl；
显示带标题的数据，内容属性为Content和Header，只能容纳一个元素；

ItemsControl族

Menu MenuBase ContextMenu ComboBox
ItemsControl ListBox ListView TabControl
TreeView Selector StatusBar -

派生自ItemsControl类的控件，用于显示列表数据，内容属性为Items或ItemsSource；

ItemsControl 对应Item Container
ComboBox ComboBoxItem
ContextMenu MenuItem
ListBox ListBoxItem
ListView ListViewItem
Menu MenuItem
StatusBar StatusBarItem
TabControl TabItem
TreeView TreeViewItem

后台数据动态显示；

HeaderedItemsControl族

和ItemsControl非常类似，派生自HeaderedItemsControl类的控件；
用于显示列表数据，附带标题，内容属性为Items、ItemsSource和Header；

Decorator族

ButtonChrome ClassicBorderDecorator ListBoxChrome SystemDropShadowChrome
Border InkPresenter BulletDecorator Viewbox
AdornerDecorator

均派生自Decorator类，起UI装饰作用，内容属性为Child，只能由单一元素充当；

TextBlock和TextBox

TextBlock TextBox
无法编辑，但可以使用丰富的排版效果 允许用户编辑
内容属性Inlines和Text 内容属性Text，内容是简单字符串

Shape族元素

Shape族元素只是视觉元素，不是控件

均派生自Shape类；
用于2D图形绘制，无内容属性；
使用Fill属性设置填充，使用Stroke属性设置边线；

Panel族元素

Canvas DockPanel Grid TabPanel
ToolBarOverflowPanel StackPanel ToolBarPanel UniformGrid
VirtualizingPanel VirtualizingStackPanel WrapPanel

均派生自Panel抽象类，主要功能是控制UI布局，内容属性为Children，内容可以是多个元素；
