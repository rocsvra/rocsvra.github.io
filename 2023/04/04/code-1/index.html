编码（1）：编码是什么

发表于 2023-04-04 更新于 2023-09-11 分类于 教程 ， 编码

编码是用来在机器和人之间传递信息的方式。

编码概述

广义上说，编码便是交流，比如英语词汇就是一种编码；
人们在沟通时使用了各种不同的编码，语言不能在纸上存储，就使用了文字，语言、文字不适合用来在黑夜中安静地传递消息，所以使用了摩尔斯码；
只要一种编码适用于其他编码所不能适用的场合，它就是一种有用的编码；

编码与组合

两个不同的事物，经过一定的组合，就可以表示所有类型的信息；

摩尔斯码是二进制码，这种编码只有两个元素——点和划，点和划的组合可以表示任意数目的码字；

布莱叶盲文和二进制码

布莱叶盲文为了便于盲人阅读发明的一种编码，布莱叶盲文中，6个凸点形成了64种可能的编码；
64中编码中的很大一部分，根据上下文的不同有着双重身份；

  1. 优先码（precedence codes）/换挡码（shift codes）
    数字标识符和取消“数字标识状态”的字母标识符，改变了后边编码的意义，从表示字母到表示数字，又从表示数字回到表示字母，它们改变着作用域内编码的含义，直到作用域结束；

  2. 逃逸码（escape codes）
    大写字母标识符“逃离”了对编码串单调的、一成不变的解析，而转入一种新的解析方法中。

电报机

19世纪初，电报的出现解决了远距离即时通讯的问题，电报机的电键被按下时，一次快速的“嘀-嗒”代表点，一次慢速的“嘀——嗒”代表划；
电报使用了二进制码，后来在无线通信中（电话、无线电、电视）中被废弃，在电脑、光盘、数字影碟、数字卫星电视广播等再次得到应用；

继电器

继电器是一个开关，但并不是由人工而是借助于电流进行开关操作的；

利用继电器可以做出令人惊奇的事情，比如，装配出一台计算机中的大部分部件；
