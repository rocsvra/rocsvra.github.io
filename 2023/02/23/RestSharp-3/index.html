RestSharp(3)：鉴权

发表于 2023-02-23 分类于 .NET

RestSharp 包括用于基本 HTTP（授权标头）、NTLM 和基于参数的系统的鉴权

SimpleAuthenticator

允许传递用户名/密码（API和密钥）作为请求方法Get/Post参数

生成url http://example.com/resource?username=foo&password=bar
请求更改为Post/Put，值将作为编码的表单值进行发送

自定义鉴权

通过实现IAuthenticator接口，并在RestClient进行注册

RestClient.Execute 调用首先执行 Authenticate 方法
Authenticate 方法通过当前正在执行的 RestRequest 提供请求数据的每一个部分（头，参数等）的访问
