小程序(18)：自定义组件

发表于 2024-08-21 更新于 2024-08-22 分类于 前端 ， 小程序

自定义组件使用

新建文件夹，右键新建Compontent；

引用组件

局部引用

[page].json，组件只能在当前被引用的页面内使用。

全局引用

app.json，组件可以在每个小程序页面中使用。

自定义组件特点

  1. [component].json
  2. [component].js 调用的是 Component() 函数，事件处理函数需要定义到 methods 节点中；

组件样式隔离

可选值 说明
isolated 默认值，启用样式隔离，自定义组件内外，使用 class 指定的样式将不会相互影响
apply-shared 页面 wxss 样式将影响到自定义组件，自定义组件 wxss 中指定的样式不会影响页面
shared 页面 wxss 样式将影响到自定义组件，自定义组件 wxss 中指定的样式也会影响页面和其他自定义组件

只有 class 选择器会有样式隔离效果，id 选择器、属性选择器、标签选择器不受样式隔离的影响；

自定义组件属性

properties 是组件的对外属性，接受外界传递到组件的数据；

数据监听器

observers 是组件的监听器，监听组件的属性/数据字段的变化，发生变化时，执行对应的回调函数；

纯数据字段

纯数据字段只能处理页面逻辑，不能用于页面渲染
