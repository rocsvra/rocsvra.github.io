小程序(17)：生命周期

发表于 2024-08-21 分类于 前端 ， 小程序

小程序的启动，表示生命周期的开始，小程序的关闭，表示生命周期的结束。

生命周期 说明
应用 小程序启动 -> 运行 -> 销毁
页面 页面创建 -> 渲染 -> 销毁

应用的生命周期函数

app.js中声明

函数 说明
onLaunch 小程序初始化完成时触发，仅一次。wx.getLaunchOptionsSync可获取参数
onShow 小程序启动、后台进入前台显示。都会触发，可以使用 wx.onAppShow 绑定监听
onHide 小程序从前台进入后台，会触发。可以使用 wx.onAppHide 绑定监听
onError 小程序发生脚本错误，或 api 调用失败，会触发 onError 并带上错误信息。可以使用 wx.onError 绑定监听

页面的生命周期函数

page.js中声明

函数 说明
onLoad 页面加载时触发，仅一次
onShow 页面显示/切入前台时触发
onReady 页面初次渲染完成时触发，仅一次，页面已经准备完成，可以和视图层进行交互
onHide 页面隐藏/切入后台时触发，如：wx.navigateTo切换至其他页面，或小程序切入后台
onUnload 页面卸载时触发。如 wx.redirectTo 或 wx.navigateBack 到其他页面

一般页面数据初始化在onLoad中完成
界面内容设置的API（如wx.setNavigationBarTitle），需要在onReady之后进行
