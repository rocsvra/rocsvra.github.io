小程序(14)：页面导航

发表于 2024-08-20 分类于 前端 ， 小程序

页面导航是指页面直接的相互跳转。

声明式导航

通过<navigator>组件实现跳转

导航到页面

属性 说明
url 页面地址，以 / 开头
open-type 跳转方式，switchTab / navigate

导航到tabBar页面，open-type 必须为 switchTab；
导航到非tabBar页面，open-type 可以为 navigate，也可以省略；

后退导航

属性 说明
open-type navigateBack
delta 数字，表示要后退的层级，默认 1

编程式导航

通过微信提供的接口实现

导航到tabBar页面

wx.switchTab(Object object) 跳转tabBar页面

参数 类型 必选 说明
url string 是 tabBar 页面路径，路径后不能带参
success function 否 成功回调函数
fail function 否 失败回调函数
complete function 否 结束回调函数，不论成功失败

导航到非tabBar页面

wx.navigateTo(Object object) 跳转非tabBar页面，参数同 wx.switchTab(Object object)

参数 类型 必选 说明
url string 是 非 tabBar 页面路径，路径后可以带参

后退导航

wx.navigateBack(Object object) 返回上一页面或多级页面

参数 类型 必选 说明
delta number 否 默认 1

导航传参

url属性中携带，参数与路径通过?分割，参数名与参数值通过=分割，多个参数通过&分割。
