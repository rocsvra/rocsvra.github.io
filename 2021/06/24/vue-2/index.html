开发环境（IDE）

运行环境（Node.js）

安装地址：官网 | 国内

查看是否安装成功

包管理器（npm/cnpm）

npm：Node Package Manager
npm 运行在node.js 上，安装node.js会同时安装npm
cnpm 是阿里搭建的一个npm在国内的镜像

模块打包器（webpack）

webpack会分析项目结构，找到js模块以及一些浏览器无法运行的拓展语言（scss,TypeScript等），将其打包为合适的格式，供浏览器使用。

创建vue项目模板（vue-cli脚手架）

创建project

基础概念

Babel：一款JavaScript转译器，可以将ES6、ES7的语法转换为现阶段浏览器可以兼容的js代码
TypeScript：JavaScript的一个超集，支持ES6标准
ESLint：编码规范标准
Router：路由
PWA：渐进式WEB应用
CSS Pre-processors：css预处理器
Vuex：状态管理

项目结构（4.0）

1.index.html 是SPA的挂载页面，该页面通过锚点来实现路由的切换

2.App.vue 是页面所有路由对应组件的容器

3.main.js 入口文件，初始化vue实例，把组件和入口页面联系起来
