Docker(3)：常用命令

发表于 2021-06-14 更新于 2023-09-11 分类于 DevOps ， docker

帮助指令

镜像指令

NAME: 镜像仓库源名称
DESCRIPTION: 镜像描述
OFFICIAL: 是否 docker 官方发布
stars: 类似 Github 里面的 star
AUTOMATED: 自动构建

REPOSITORY：镜像仓库源
TAG：镜像标签，同意仓库源可以有多个TAG，代表不同版本，默认latest
IMAGE ID：镜像ID
CREATED：镜像创建时间
SIZE：镜像大小

创建一个容器 docker run
更新容器 apt-get update
退出容器 exit
提交容器副本 docker commit
查看新镜像 docker images

设置id为860c279d2fec的镜像，标签为dev

容器

-d 后台运行
-P 容器内部端口随机映射到主机端口
-p 容器内部端口绑定到指定主机端口
–rm 容器退出自动清理容器内部的文件系统
-h 设定容器的主机名，它会被写到容器内的 /etc/hostname 和 /etc/hosts

CONTAINER ID: 容器 ID
IMAGE: 使用的镜像
COMMAND: 启动容器时运行的命令
CREATED: 容器的创建时间
STATUS: 容器状态(created | restarting | running或Up removing | paused | exited | dead)
PORTS: 容器的端口信息和使用的连接类型（tcp\udp）
NAMES: 容器名称
