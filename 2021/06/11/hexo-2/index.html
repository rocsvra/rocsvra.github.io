Github Pages简介

github提供了pages功能，分别是user pages和project pages，区别如下——

user pages project pages
账号唯一 repo唯一
repo名必须为[username].github.io, 页面在master上 repo名不限定，页面在分支gh-pages下
http(s)://[username].github.io http(s)://[username].github.io/[projectname]

我们可以利用Github Pages的static web server功能，host Hexo生成的静态博客文件

设置个人域名（可选）

1.买一个域名，添加解析，解析路线选默认，把github的服务器地址添加进去
2.github -> settings -> Custom domain -> 你的域名
3.本地source文件夹创建CNAME文件（无后缀），写上你的域名
