小程序(1)：简介

发表于 2021-09-28 更新于 2021-09-29 分类于 小程序

简介

小程序可以看做只能用微信打开和浏览的网站。

小程序和网页技术模型是一样的，用到的javascript和css也是一样的，只是html标签被稍晚修改成了wxml标签，小程序页面本质就是网页。
但小程序页面不支持浏览器，所有浏览器api都不能使用，只能用微信提供的api，底层变了。

优势

微信的功能（拍照、扫描、支付等），小程序大部分能够使用，微信提供了各种封装好的api，不用自己去实现，也不用考虑ios和安卓的平台差异，调用代码是一样的。
小程序不用考虑用户注册和登录，微信的用户自动成为小程序的用户。

注册

免费注册地址：微信公众平台
注册成功后登录，可以在开发设置中得到AppID和AppSecret

开发工具

下载地址：小程序开发工具
这个工具是必须的，只有这个工具才能运行和调试小程序代码

  1. 新建项目目录
  2. 新建脚本app.js
  3. 新建配置文件app.json

    每个页面有一个自己目录，小程序会加载pages/home里的home.js，js后缀名可以省略，所以完整的加载路径为pages/home/home，home.js的文件名可以随便起，但习惯上跟页面目录同名

  4. 目录下新建文件home.js
  5. 新建home.wxml

    wxml是微信页面标签语言

  6. 项目结构

    项目两部分组成，描述整体程序的顶层app脚本，以及描述各个页面的page脚本
    这是最基本的小程序结构，所有的小程序项目，都不过是它的扩展
    用小程序开发工具导入项目目录，可以看运行结果

  7. app.json添加window属性，开发工具里可以看到小程序里导航栏的变化
